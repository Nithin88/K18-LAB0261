# Lab 1: The Portal Record
## Goal
In this lab, we will create a new portal record and a new theme.

## Create the Portal
1.	Open your web browser and navigate to the unique instance URL provided to you.
2. Login as **admin**.
3.	In the application navigator, choose **Studio** and pick the **Idea Portal** application. This has been pre-loaded onto your instance.
4. In Studio, click on **Idea Portal**, which will appear under the heading **Service Portals**. This is another record which has been pre-loaded on your instance.
6.	Notice the **URL Suffix** to access this portal is **ideas**.
7.	Download **[ServiceNow.png](resources/lab-01/ServiceNow.png)** and upload it to the **Logo** field.
8.	Press **Save** and close the window.
9.	This portal is associated with a Idea Portal theme (included OOTB on your instance) click on the icon shown below to open the theme record and make the suggested changes

![Theme](images/lab-01/01-theme.png)

| Field         | Value                                     |
|---------------|-------------------------------------------|
| Header        | Stock Header *(we will use the OOB header)* |
| CSS Variables | *Contents of [theme.css](scripts/lab-01/theme.css)*|
